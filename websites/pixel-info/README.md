# Qu'est ce que Pixel-Info ? 

![logo](https://github.com/OrAxelerator/web/blob/main/websites/pixel-info/Pixel-info_logo.png?raw=true)

Pixel-Info est un petit site que j’ai créé pour m’entraîner à JavaScript et à manipuler une base de données (firebase). C’est aussi un espace où je peux créer et partager des articles sur différents sujets.
À terme, j’aimerais même permettre à d’autres personnes d’ajouter leurs propres articles sur Pixel-Info, si j’intègre cette fonctionnalité ..
